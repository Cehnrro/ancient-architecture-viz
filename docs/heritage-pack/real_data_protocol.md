# 真实数据采集规范（v1）

## 目标
- 只接收可核验条目。
- 每条记录至少 1 条可访问来源链接（sourceRefs）。
- 允许先用公开百科建立候选池，但进入答辩版前需补充至少 1 条权威来源（古籍影印/出版社/政府或高校机构页）。

## 来源分级
- A级：国家/高校/出版社/文博机构/古籍数据库。
- B级：学术百科、专题数据库。
- C级：大众百科（仅作线索，不作最终唯一依据）。

## 入库门槛
- 著作：必须是真实存在的文献或典籍（可佚失但须有可靠记载）。
- 人物：必须有稳定史料或学术条目可追溯。
- 关系边：必须有 evidenceRef；无证据不得入主图。

- 当前数据为“可核验首批”，下一轮优先补 A 级来源覆盖率。
